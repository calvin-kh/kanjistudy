# Plan: 칸지공부 (KanjiStudy) v1.0

## 1. 개요

| 항목 | 내용 |
|------|------|
| 프로젝트명 | 칸지공부 (KanjiStudy) |
| 회사 | kamuse.inc. |
| 패키지 | com.kanjistudy.app |
| 목표 | JLPT N5~N1 상용한자 2,136자를 효과적으로 학습할 수 있는 모바일 앱 |
| 플랫폼 | Android / iOS / Web |
| 배포 | Google Play Store (우선), App Store (추후) |

## 2. 배경 및 동기

- 일본어 학습자에게 한자는 가장 큰 진입장벽
- 기존 한자 앱은 대부분 영어 기반이거나 유료 구독 모델
- **한국어 UI**로 된 무료 오프라인 한자 학습 앱이 부재
- JLPT 시험 준비에 최적화된 레벨별 학습 도구 필요

## 3. 핵심 요구사항

### 3-1. 기능 요구사항 (Functional Requirements)

| ID | 기능 | 우선순위 | 설명 |
|----|------|----------|------|
| FR-01 | 한자 사전 | P0 | 2,136자 한자 데이터 (뜻, 음독, 훈독, 예제, 획수, 학년) |
| FR-02 | JLPT 레벨별 분류 | P0 | N5~N1 레벨별 한자 목록 필터링 |
| FR-03 | 한자 상세 보기 | P0 | 개별 한자의 모든 정보 + 스와이프 네비게이션 |
| FR-04 | 검색 | P0 | 한자, 읽기(히라가나/카타카나), 의미 퍼지 검색 |
| FR-05 | 즐겨찾기 | P1 | 한자 즐겨찾기 등록/해제/목록 |
| FR-06 | 퀴즈 | P1 | 한자→뜻, 뜻→한자, 한자→읽기 3가지 퀴즈 |
| FR-07 | 학습 진도 | P1 | 미학습/학습중/완료 3단계 상태 관리 |
| FR-08 | 학습 통계 | P1 | 퀴즈 정답률, 레벨별 진도 시각화 |
| FR-09 | 설정 | P2 | 테마(라이트/다크/시스템), 일일 목표, 데이터 초기화 |
| FR-10 | 약관 동의 | P0 | 최초 실행 시 이용약관/개인정보처리방침 동의 |
| FR-11 | 랜딩페이지 | P2 | 앱 소개 + 다운로드 링크 웹페이지 |

### 3-2. 비기능 요구사항 (Non-Functional Requirements)

| ID | 요구사항 | 설명 |
|----|----------|------|
| NFR-01 | 완전 오프라인 | 서버 없음, 로그인 없음, 인터넷 불필요 |
| NFR-02 | 개인정보 미수집 | userId 없음, 모든 데이터 로컬 저장 |
| NFR-03 | 다크/라이트 모드 | 시스템 설정 연동 + 수동 전환 |
| NFR-04 | 한국어 UI | 모든 사용자 대면 텍스트 한국어 |
| NFR-05 | 빠른 실행 | 앱 시작 시 로딩 최소화 |

## 4. 대상 사용자

| 페르소나 | 설명 |
|----------|------|
| JLPT 수험생 | N5~N1 시험을 준비하는 한국인 학습자 |
| 일본어 초급자 | 기본 한자부터 체계적으로 공부하고 싶은 사용자 |
| 복습 목적 | 알고 있는 한자를 퀴즈로 점검하고 싶은 사용자 |

## 5. 범위 (Scope)

### In-Scope (v1.0)
- JLPT 상용한자 2,136자 데이터
- 한자 사전, 검색, 즐겨찾기
- 3종 퀴즈 (한자→뜻, 뜻→한자, 한자→읽기)
- 학습 진도 관리 + 통계
- 다크/라이트 테마
- 약관 동의 화면
- 랜딩페이지 (Vercel)
- Google Play Store 배포

### Out-of-Scope (v1.0)
- 사용자 계정 / 클라우드 동기화
- SRS (Spaced Repetition System)
- 한자 필순 애니메이션
- App Store 배포
- 광고 / 인앱결제
- 다국어 지원

## 6. 일정

| 단계 | 기간 | 산출물 |
|------|------|--------|
| 기획 + 설계 | 1일 | Plan, Design 문서 |
| 초기 구현 | 1일 | 전체 기능 구현 (초기 커밋) |
| 버그 수정 + 폴리싱 | 1일 | 다크모드, 텍스트 색상, 아이콘 수정 |
| 스토어 배포 | 1일 | Play Store 빌드 + 업로드 |
| 기능 추가 | 진행 중 | 스와이프 네비게이션, UI 개선 |
